부팅이 안될 때 응급 조치
[cmos setup을 Default 값으로 하기]





컴퓨터 전원 스위치를 켜고 곧바로 F2키나 Delete키를 반복해서 눌러 줍니다.
그러면 아래 화면과 비슷한 cmos setup창이 뜹니다.



(컴퓨터 부팅이 안될 때 cmos setup을 Default 값으로 변경하기 위하여 cmos setup을 실행한 화면)





여기서 오른쪽 방향키를 이용해서 맨 오른 쪽 'Exit'메뉴로 이동합니다.





(컴퓨터 부팅이 안될 때 cmos setup을 Default 값으로 변경하기 위하여 'Exit'메뉴로 이동하기)





'Exit'메뉴로 이동한 화면입니다. 이대로 엔터키를 치시면 안됩니다. 맨 아래 'Load Setup Default'를 선택하셔야 합니다.





(컴퓨터 부팅이 안될 때 cmos setup을 Default 값으로 변경하는 화면)





여기서 엔터키를 치시면 디폴트 값을 로딩할 것이냐고 묻는데 'OK'누르고 나오시면 됩니다.
그러면 cmos setup을 Default값으로 해서 부팅을 하게 됩니다.
